[[/Glm/Macro/Verbose]] -- Verbose output message

# Synopsis

GLM:

~~~
#verbose <message>
~~~

# Description

The `#verbose` macro outputs a message to the verbose stream.

# Examples

~~~
#verbose This is a debugging message
~~~
