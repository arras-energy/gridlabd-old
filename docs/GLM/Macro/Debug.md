[[/Glm/Macro/Debug]] -- Debug output message

# Synopsis

GLM:

~~~
#debug <message>
~~~

# Description

The `#debug` macro outputs a message to the debugging stream.

# Examples

~~~
#debug This is a debugging message
~~~
