[[/Glm/Macro/Option]] -- Macro to specify a command-line option

# Synopsis
GLM:
~~~
#option <keyword>
~~~
Shell:
~~~
bash$ gridlabd --<option> ...
~~~

# Description

The `#option` macro is equivalent to specifying the `<option>` on the command line.

# Examples

The following toggles verbose output
~~~
#option verbose
~~~
