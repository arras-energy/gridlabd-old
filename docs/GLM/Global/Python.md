[[/GLM/Global/Python]] -- Global expansion of python expression

# Synopsis
GLM:
~~~
${PYTHON <expression>}
~~~

# Description

The python expression is inserted into the GLM file at the location of the global expansion.

# Example

The following example prints the python list `[1,2,3]`:
~~~
#print ${PYTHON [1,2,3]}
~~~
