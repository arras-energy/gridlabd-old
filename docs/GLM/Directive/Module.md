[[/GLM/Directive/Module]] -- GLM module load directive

# Synopsis

GLM:

~~~
module NAME;
~~~

# Description

The `module` load directive is used to load a GridLAB-D or Python modules into the solver framework.  Modules provide classes and event handlers.
