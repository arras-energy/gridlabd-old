[[/Global/Initializeonly]] -- Initialize only enable flag

# Synopsis

GLM:

~~~
#set Initializeonly=FALSE
~~~

Shell:

~~~
bash$ gridlabd -D Initializeonly=FALSE
bash$ gridlabd --define Initializeonly=FALSE
~~~

# Description

Initialize only enable flag

# Example

~~~
#set compileonly=FALSE
~~~
