[[/Subcommand/Valgrind]] -- Run gridlabd with valgrind

# Synopsis

~~~
bash$ export VALGRIND_OPTIONS=...
bash$ gridlabd valgrind [<options> ...]
~~~

# Description

This subcommand enables the valgrind tool if it is installed on the host system.
