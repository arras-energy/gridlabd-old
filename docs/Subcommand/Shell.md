[[/Subcommand/Shell]] -- Open GridLAB-D shell environment

# Synopsis

Shell:

~~~
sh$ gridlabd shell
~~~

# Description

The `shell` subcommand opens a shell in the GridLAB-D environment. This can be used to run Python with the active python `venv` used for GridLAB-D.
