IMPORTANT NOTICE: This installer is designed to install HiPAS GridLAB-D on a new system image such as wls, ec2, or docker. Do not use this on an existing system unless you are ok with significant changes to your system's python infrastructure and system build tools.

To install HiPAS GridLAB-D on your Linux or Mac system, type the following command at a shell prompt:

bash$ curl -sL http://install.gridlabd.us/install.sh | sudo /bin/bash

If your system is not supported by this installer, please use the manual installation method provided with the HiPAS GridLAB-D source code.
